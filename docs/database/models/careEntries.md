# CareEntry

## Overview

Represents a single care event recorded for a plant (for example watering, fertilizing, pruning, repotting).

Care entries are stored as a subcollection under a plant and are used for plant history and watering schedule inference.

## Fields

| Field         | Type   | Optional | Description                                                    |
| ------------- | ------ | -------- | -------------------------------------------------------------- |
| id            | string | no       | Unique identifier generated by Firebase                        |
| careType      | string | no       | Type of care event (for example `water`, `fertilize`, `prune`) |
| date          | Date   | no       | Date/time when the care event happened                         |
| notes         | string | yes      | Free-form notes; defaults to `""`                              |
| otherCareType | string | yes      | Optional custom care label; defaults to `""`                   |

## Relationships

- **CareEntry → Plant** (many-to-one)
  Each care entry belongs to one plant.

- **CareEntry → User** (many-to-one, indirect via Plant)
  Each care entry is scoped to a user through its parent plant path.

## Firestore Storage

`/users/{userId}/plants/{plantId}/careEntries/{careEntryId}`

## Constraints and Rules

- `careType` must be a non-empty string.
- `date` must be a valid JavaScript `Date`.
- `notes` defaults to `""` when omitted.
- `otherCareType` defaults to `""` when omitted.
- Care entries with `careType` starting with `"water"` are treated as watering events by service logic and can trigger plant watering metadata recalculation.

## Side Effects (Water Entries)

When a care entry is recorded as water-related (`careType` starts with `"water"`), the service may update these parent Plant fields:

- `lastWateredDate`
- `secondLastWateredDate`
- `inferredWateringFrequency`
- `nextWateringDate`

## Example Document

```json
{
  "id": "entry-abc123",
  "careType": "water",
  "date": "2026-02-24T09:30:00Z",
  "notes": "Watered until slight drainage",
  "otherCareType": ""
}
```
